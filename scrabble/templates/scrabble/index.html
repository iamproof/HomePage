Scrabble Dictionary

English scrabble dictionary. Just type letters that you need to use, and program display available words from english dictionary from http://www-01.sil.org/linguistics/wordlists/english/. You need to remember that results can be different depend on the dictionary that is being used, so it's normal that other dictionarys show slightly different results.
If you are interested in the "view.py", and the function I use, You can look at the repository, here.

Additional Information:
First version of this algorythm needed alomost 5.5 minutes to display results.
Second version needed up to 30 seconds, and the 3rd version of algorythm need 0.5 second to display results.
